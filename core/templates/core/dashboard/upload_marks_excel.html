Excel format guide

Column Description
Roll/Student ID Student roll number or full name (required — used to find the student)
Student Name Optional — only for human reference
Subject(s) Comma-separated subjects, e.g. Mathematics,English,Science
Marks Obtained Comma-separated marks matching subject order, e.g. 85,78,90
Total Marks Per-subject totals e.g. 100,100,50 — or one value applies to all subjects
4 Sheets inside your Excel file:
1st Terminal 2nd Terminal 3rd Terminal Final Terminal

Each sheet must have the same 5 columns. Sheets with no data rows are simply skipped.
Tips: Subject names must match the ones configured in the system (case-insensitive). Repeat the same student on each terminal sheet as needed — marks for each terminal are stored independently.
